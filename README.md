# 智能拆词工具

本项目是一个本地运行的中文拆词小工具，支持自动分词、词语选择与一键复制，并提供夜间/白天模式切换。

## 功能特性

- 自动拆词与统计（总词数、已选择、字符数）
- 选词默认多选，支持单击选择/取消选择
- 双击词语可直接复制单个词
- 一键复制选中词语
- 夜间/白天模式切换
- 纯本地运行，无需后端服务

## 使用方式

1. 打开 `word-splitter.html`。
2. 在输入框粘贴或输入文本。
3. 点击“开始分词”。
4. 单击选择多个词语，点击“复制选中”。
5. 需要单个词语时，可双击词语直接复制。

## 拆词规则（升级版）

- 识别 URL：例如 `https://example.com/path?a=1`
- 识别邮箱：例如 `name+tag@mail.com`
- 识别话题/提及：`#topic`、`@user`
- 识别数字（含小数与千分位）：`12.5`、`1,000,000`
- 识别英文复合词：`state-of-the-art`、`snake_case`、`don't`
- 中文连续字符按整段输出
- 其他标点/符号保留为独立 token
